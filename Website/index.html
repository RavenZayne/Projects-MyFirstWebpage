Main use for text

Text are mainly used for description, and explanations on a website these tags are used to add the text and edit them.

  1. < P > is mainly used for paragrahps, it is the main tag used for adding text.

    2. Paragraphs are usually represented in visual media as blocks of text separated from adjacent blocks by blank lines and/or first-line indentation, but HTML paragraphs can be any structural grouping of related content, such as images or form fields.




Basic use for editing text

  1. <H1> is used for the text size, it could go from H1 to H6

    2. The H1 tag is an HTML heading that's most commonly used to mark up a web page title. Most websites use CSS to make the H1 stand out on the page compared to lesser headings like H2, H3, etc.

  2. < Strong >

    3. The Strong Importance element. The <strong> HTML element indicates that its contents have strong importance, seriousness, or urgency. Browsers typically render the contents in bold type.

  3. <I> defines a part of text in an alternate voice or mood.

    4. The content inside is typically displayed in italic. The <i> tag is often used to indicate a technical term, a phrase from another language, a thought, a ship name, etc.

  4. <small> defines smaller text (like copyright and other side-comments).

    5. This tag is not deprecated, but it is possible to achieve richer (or the same) effect with CSS.

  5. <abrr title="Yes"> example: You're Is for abbrevations

    6. The title attribute may be used to provide an expansion of the abbreviation. The attribute, if specified, must contain an expansion of the abbreviation, and nothing else.




Important tags

  1. <html> An HTML tag is a piece of markup language used to indicate the beginning and end of an HTML element in an HTML document.

    2. As part of an HTML element, HTML tags help web browsers convert HTML documents into web pages.

  2. <Head> The <head> element is a container for metadata (data about data) and is placed between the <html> tag and the <body> tag.

    3. Metadata is data about the HTML document. Metadata is not displayed. Metadata typically define the document title, character set, styles, scripts, and other meta information.

  3. <Body> The <Body> defines the document's body.

    4. The element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc. Note: There can only be one <body> element in an HTML document.
